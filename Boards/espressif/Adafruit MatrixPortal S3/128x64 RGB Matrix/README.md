# 128x64 4-panel display

### Hardware
- 4x 64x32 5mm Pitch RGB Matrix Panels
- Adafruit MatrixPortal S3
- Adafruit BME688 I2C sensor module for temp, humidity, pressure, & VOC
- 2x 20cm IDC cables
- 1x 30cm IDC cable
