# Airlift Online Weatherstation
- Just like my offline weatherstation except this one uses the ESP32 Airlift Featherwing to pull data from OpenWeatherMaps.org
- Data in white is from local sensors and data in blue is JSON parsed data from OpenWeatherMaps 2.5 API

# Hardware Used
- Adafruit NRF52840 Bluefruit Sense Feather
- Adafruit ESP32 Airlift Featherwing
- Adafruit 3.5" TFT Featherwing

# Libraries Used
- neopixel
- pwmio
- adafruit_display_text
- adafruit_bitmap_font
- adafruit_esp32spi
- adafruit_hx8357
