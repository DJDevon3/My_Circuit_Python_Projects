# Dragon Mask Halloween 2022
A 4x microcontroller halloween mask illumination project by DJDevon3

### Hardware Used:
- Adafruit Gemma M0 (2x 12 pixel rings)
- Adafruit QT Py ESP32-S2 (nood lighting)
- Adafruit Feather M4 Express + PropMaker Featherwing (scepter)
- Adafruit Feather ESP32-S3 (GC9A01 LCD eyes)
